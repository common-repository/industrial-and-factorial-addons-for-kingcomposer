== Installation ==

Installing this plugin as regular wordpress plugin. First you need to install King Composer Page Builder plugin then install Essential WPBakery Page Builder.

This addon plugin works like other page builder addon plugin. Go to new page then click on WPBakery Page builder then click on Add Element you should see VC Addons menu as Name "VC Addons" on Menu Top, click on it you will see shortcode item list.
